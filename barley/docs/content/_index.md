# Barley UI

**Sharp. Minimal. Semantic.**

Barley is a lightweight (~8KB gzipped), zero-dependency UI library designed for modern web applications. It focuses on semantic HTML, minimal class usage, and a sharp, monochromatic aesthetic with a touch of gold.

## Key Features

- **Zero Dependencies**: Pure CSS and Vanilla JS.
- **Semantic**: Styles native elements (`<button>`, `<dialog>`, `<details>`) directly.
- **Lightweight**: ~8KB gzipped (CSS + JS).
- **Sharp Design**: Square corners, high contrast, clean typography.
- **Dark Mode**: Automatic support via `prefers-color-scheme`.
