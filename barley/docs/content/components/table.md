Styles for tabular data.

## Usage

Styles standard HTML tables.

{% demo() %}
```html
<table>
  <thead>
    <tr>
      <th>Name</th>
      <th>Role</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>Alice</td>
      <td>Admin</td>
    </tr>
    <tr>
      <td>Bob</td>
      <td>User</td>
    </tr>
  </tbody>
</table>
```
{% end %}
