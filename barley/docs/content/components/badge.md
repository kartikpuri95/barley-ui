Small status indicators.

## Usage

Use `data-role="badge"`.

{% demo() %}
```html
<span data-role="badge">Default</span>
<span data-role="badge" data-variant="primary">Primary</span>
<span data-role="badge" data-variant="success">Success</span>
```
{% end %}

## Variants

Supports all standard color variants: `primary`, `secondary`, `accent`, `success`, `warning`, `danger`.
